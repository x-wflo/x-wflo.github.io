Unity的安装

下载

Unity中国官网：unity.cn/

该站的语言是中文，比起在原官网下载速度快了不少，对没有梯子的小伙伴来说友好不少。

在该站中我们需要下载的东西有两个：

  • Unity Installer(根据自己环境选择mac/win)

  • Unity Hub

    Unity Hub是用来管理你的Unity的，有了它你可以同时使用多个不同版本的Unity。但是即使你只需要一个版本的Unity也需要下载Unity Hub，如果不下载在安装完Unity后的第一次打开时，Unity会提醒你安装Hub。
