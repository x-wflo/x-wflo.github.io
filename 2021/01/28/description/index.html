搭建个人博客了

2021-01-28
心情随笔

乘着寒假来了，做了一直想做的事情：把个人博客搭起来了。

一直以来也想找一个地方堆积一下自己的作业。

虽然个人技术力也不是很高，在高手如云的本专业和本院中只能算是能力平平，但完成作业时候的成就感也是真实的。在痛苦的自学中，成就感也是让我从上个生不如死的学期中撑下来的原因之一。

这个博客中，也许会放一些课程作业，也会放一些心情随笔，也会放一点作品归档，也许会勤更，也许会废弃，至少在这一秒我是想要把它经营好的。

使用的是hexo+GitHub仓搭建，hexo主题用的是ocean，非常好看又简约的一个主题，评论系统用的是Valine。
